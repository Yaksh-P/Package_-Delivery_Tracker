# Project Summary

## Project Title

Delivery Package Tracker

## Project Overview

The Delivery Package Tracker is a full-stack web application designed to help users create, manage, and track package deliveries. The system allows users to add package information, view package status, update delivery details, and monitor tracking history.

## Target Users

Customers and delivery staff

## Core Features

1. Add a new package
2. View all packages
3. Update package details and delivery status
4. Delete package records
5. View package tracking updates
